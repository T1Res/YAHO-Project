로그인 / 회원가입

회원가입 이용 방법

  • 사이트 홈 상단 메뉴 오른쪽에서 회원가입 탭 선택
  • 회원가입 페이지로 이동
  • 필수 입력 사항 기입 후 등록 버튼 선택
  • 회원가입 완료

로그인 이용 방법

  • 사이트 홈 상단 메뉴 오른쪽에서 로그인 탭 선택
  • 로그인 페이지로 이동
  • 로그인 아이디, 비밀번호 입력 후 로그인 버튼 선택
  • 로그인 성공
  • 로그인 실패 시 아이디 또는 비밀번호 오류 안내 후 재시도 가능

프로필 및 회원정보 수정

  • 로그인 후 사이트 홈 상단 오른쪽의 프로필 이미지 선택
  • 프로필 사진 아래 메뉴에서 '프로필정보' 탭 선택
  • 수정 후 비밀번호 입력, 수정 버튼 선택
  • 회원정보 수정 완료

사이트 이용 안내

홈 / 메인 페이지

  • 추천 애니 목록 확인
  • 오늘의 애니 목록 제공
  • 애니 OST 시청 (YouTube 링크 포함)
  • 애니 선택 시 상세정보, 평점, 리뷰 확인 가능

전체 애니 페이지

  • 전체 애니 목록 제공
  • 정렬 기능으로 애니 정렬 가능
  • 태그별 애니 검색 가능
  • 검색바를 통한 검색 지원
  • 애니 선택 시 평점 및 리뷰 확인

방영중 애니 페이지

  • 현재 방영 중인 애니 시간표 제공
  • 애니 선택 시 상세정보, 평점, 리뷰 확인
